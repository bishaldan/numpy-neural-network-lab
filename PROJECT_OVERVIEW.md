# numpy-neural-network-lab Overview

## Problem

`numpy-neural-network-lab` started as a simple "neural network from scratch" exercise, then evolved into a stronger AI/ML engineering portfolio piece.

The goal is to show both:

- understanding of neural network math
- engineering discipline around training, evaluation, Docker, testing, and presentation

## Why this project matters

Many beginner ML repositories stop at a notebook. This repository pushes further by packaging the model into a reproducible workflow with validation tracking, regularization, experiment comparison, inference, and a lightweight demo.

## Key portfolio signals

- NumPy-only model implementation
- validation-aware training loop
- dropout, L2 regularization, learning-rate decay, and early stopping
- experiment runner with summary comparisons
- Docker-based reproducibility
- Streamlit demo for fast reviewer interaction
- GitHub Actions CI for clean cloning and testing
